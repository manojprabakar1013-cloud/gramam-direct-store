# Gramam Direct E-commerce Website

A responsive multi-page static e-commerce front end for **Gramam Direct**.

## Included pages
- Home
- Shop / product listing
- Product details
- Cart
- Checkout
- Login
- Register
- User dashboard
- My orders
- Our story / About
- Contact

## Features
- Responsive earthy rural-inspired branding
- Search, category filter and price sorting
- Working cart using browser localStorage
- Demo checkout and order history
- Demo login/register and dashboard
- Local SVG product artwork (no external image dependency)

## Run locally
Open `index.html` directly, or serve the folder using any static web server.

## Deploy on GitHub Pages
1. Upload all files to a public GitHub repository.
2. Open **Settings → Pages**.
3. Select **Deploy from a branch**.
4. Choose `main` and `/root`, then save.

## Before going live
- Replace demo contact details.
- Replace sample products and prices.
- Connect a backend/database for real users and inventory.
- Connect a payment gateway such as Razorpay or Stripe.
- Add shipping, return, privacy and terms policies.


## Brand theme

Organic Farm Theme using Deep Leaf Green, Warm Earthy Brown, and a clean white background to convey fresh produce, natural soil, eco-friendly sourcing, and direct-from-farm trust.
